本周AI代理研究从物理自校正、小模型边缘化、认知问责到通信协议标准化的系统化突破，标志着代理技术正从“能完成任务”加速迈向“可靠、可解释、可互操作”的新阶段。

  • AdsMind 通过将大模型生成与机器学习力场物理松弛闭环，使吸附构型发现成功率达到近乎100%，同时将计算成本降低约14倍，证明了物理反馈对消除LLM代理错觉的关键作用。
  • 微软发布 MagenticLite 等系统，用“专门模型+编排”架构让代理在浏览器和本地文件系统间自主执行任务，首次将复杂代理能力压缩进小模型，为边缘隐私智能应用铺路。
  • 结构化认知循环（SCL）将认知、记忆、控制与行为彻底分离，让每一步动作都经过前置检查和授权，在旅行规划等任务上成功率提升至86.3%，并首次引入认知问责机制。
